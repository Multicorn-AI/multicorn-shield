    Interface ScopeRegistry

    A registry of service definitions that maps service names to their capabilities. Pre-populated with built-in services and extensible with custom ones.

    Created via createScopeRegistry.

    Example

    const registry = createScopeRegistry();

    registry.register({
      name: "analytics",
      description: "Internal analytics",
      capabilities: ["read"],
    });

    registry.has("analytics"); // true
    registry.isValidScope({ service: "analytics", permissionLevel: "read" }); // true
