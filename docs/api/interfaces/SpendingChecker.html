    Interface SpendingChecker

    Client-side spending checker for pre-validating agent transactions.

    Tracks cumulative spend in memory and enforces per-transaction, per-day, and per-month limits. Provides instant feedback before API calls.

    Important: This is a client-side cache that resets on page reload. The server is the source of truth and performs final validation.

    Example: Basic usage

    const checker = createSpendingChecker({
      limits: {
        perTransaction: 20000,  // $200.00
        perDay: 100000,         // $1,000.00
        perMonth: 500000,       // $5,000.00
      },
    });

    // Pre-check before executing action
    const result = checker.checkSpend(5000); // $50.00
    if (!result.allowed) {
      console.error(`Blocked: ${result.reason}`);
      return;
    }

    // Execute the action...
    await executeAction();

    // Record successful spend for tracking
    checker.recordSpend(5000);

    Example: Checking remaining budget

    const current = checker.getCurrentSpend();
    console.log(`Spent today: $${(current.daily / 100).toFixed(2)}`);
    console.log(`Spent this month: $${(current.monthly / 100).toFixed(2)}`);

    checkSpend

    • checkSpend(amountCents: number): SpendingCheckResult

      Check whether a proposed spend would be allowed under the configured limits.

      This is a read-only pre-check that does NOT record the spend. Call recordSpend after a successful transaction to update tracking.

      Checks are performed in order:

      1. Per-transaction limit
      2. Per-day cumulative limit
      3. Per-month cumulative limit

      The check fails at the first violated limit with a specific error message.

      Parameters

      • amountCents: number

        The proposed spend amount in integer cents.

      Returns SpendingCheckResult

      A SpendingCheckResult indicating whether the spend is allowed.

      Throws

      If the amount is negative or not an integer.

      Example

      const result = checker.checkSpend(5000); // $50.00
      if (result.allowed) {
        console.log(`OK. Remaining today: $${(result.remainingBudget.daily / 100).toFixed(2)}`);
      } else {
        console.error(result.reason);
      }

    • recordSpend(amountCents: number): void

      Record a successful spend for cumulative tracking.

      Call this after an action has been successfully executed and charged. This updates the daily and monthly cumulative spend totals.

      Parameters

        The actual spend amount in integer cents.

      Returns void

      Throws

      If the amount is negative or not an integer.

      Example

      if (result.allowed) {
        await executeAction();
